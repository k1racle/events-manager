# Events Manager

**Events Manager** — это кастомный плагин для WordPress, который добавляет тип записей «События» с расширенными полями (дата, город, место, описание, тип мероприятия, изображение). Плагин предоставляет удобный шорткод для вывода списка событий с фильтрацией по дате, городу и типу, AJAX-подгрузкой и модальным окном с подробной информацией и картой (Яндекс или Google на выбор).

## Возможности

- ✅ Кастомный тип записей `event` с поддержкой миниатюр.
- ✅ Таксономия «Тип мероприятия» для классификации событий.
- ✅ Метабокс в админке для ввода даты, города, места и описания.
- ✅ Шорткод `[events_list]` с настраиваемыми параметрами.
- ✅ Фильтрация событий по дате (будущие, прошедшие, на этой/следующей неделе, в этом/следующем месяце, все).
- ✅ Фильтрация по городу (список формируется автоматически из добавленных событий).
- ✅ Фильтрация по типу мероприятия.
- ✅ AJAX-подгрузка следующих страниц (кнопка «Показать больше»).
- ✅ Модальное окно с детальным описанием события и картой (открывается по клику на карточку).
- ✅ Выбор провайдера карт (Яндекс.Карты или Google Карты) в настройках.
- ✅ Кастомизация основного цвета элементов через настройки.
- ✅ Защита AJAX-запросов с помощью `wp_nonce`.
- ✅ Учёт часового пояса WordPress при работе с датами.
- ✅ Адаптивный дизайн (сетка 3 колонки на десктопе, 2 на планшете, 1 на мобильном).

## Требования

- WordPress 5.0 или выше.
- PHP 7.2 или выше.

## Установка

1. Скачайте архив с плагином или склонируйте репозиторий в папку `/wp-content/plugins/events-manager/`.
2. Активируйте плагин через меню «Плагины» в админке WordPress.
3. Перейдите в **Настройки → Events Manager** для выбора провайдера карт и основного цвета.
4. В меню появится пункт **«События»** — добавьте несколько событий, заполнив все поля.
5. Также появится таксономия **«Типы событий»** — создайте нужные типы (например, Концерт, Лекция, Кино).

## Использование шорткода

Для вывода списка событий используйте шорткод `[events_list]` в любой записи, странице или виджете.

### Атрибуты шорткода

| Атрибут         | По умолчанию | Описание |
|-----------------|--------------|----------|
| `count`         | 3            | Количество событий, отображаемых за одну загрузку. |
| `show_map`      | 1            | Показывать карту в модальном окне (1 — да, 0 — нет). |
| `show_image`    | 1            | Показывать изображение события в карточке (1 — да, 0 — нет). |
| `show_filters`  | 1            | Показывать панель фильтров над списком (1 — да, 0 — нет). |
| `map_provider`  | из настроек  | Провайдер карт: `yandex` или `google`. Переопределяет значение из настроек. |

### Примеры

- `[events_list]` — базовый вывод с параметрами по умолчанию.
- `[events_list count="6" show_map="0"]` — 6 событий на страницу, без карты.
- `[events_list map_provider="google"]` — принудительное использование Google Карт.
- `[events_list show_filters="0"]` — скрыть фильтры.

## Фильтрация событий

Если атрибут `show_filters="1"` (по умолчанию), над списком отображается панель с тремя выпадающими списками:

- **Фильтр по дате**: будущие, прошедшие, на этой неделе, на следующей неделе, в этом месяце, в следующем месяце, все.
- **Фильтр по городу**: все города или конкретный город (список пополняется автоматически при добавлении событий).
- **Фильтр по типу**: все типы или конкретный тип мероприятия.

После выбора значений нажмите кнопку **«Применить фильтры»** — список обновится без перезагрузки страницы (AJAX).

## Настройки плагина

В разделе **Настройки → Events Manager** доступны:

- **Провайдер карт по умолчанию** — выбор между Яндекс.Картами и Google Картами.
- **Основной цвет** — цвет кнопок и акцентных элементов (кнопка «Применить фильтры», «Показать больше», граница селектов при наведении, кнопка «Подробнее»).

## Внешний вид и поведение

- События отображаются в виде карточек с изображением (если включено), названием, типом, датой, городом и местом.
- Карточка полностью кликабельна — по клику открывается модальное окно с подробным описанием и картой.
- В модальном окне показывается заголовок, тип, дата, город, место, описание и карта (если включена).
- Кнопка **«Показать больше»** загружает следующую порцию событий (без перезагрузки). Если событий больше нет, кнопка исчезает.
- Все AJAX-запросы защищены nonce.
